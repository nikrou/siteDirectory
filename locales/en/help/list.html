That plugin allow you to manage your sites.

Usage

  1. Activate plugin (Parameters tab)
  2. Add a type (Types tab)
  3. Add a theme (Themes tab)

Parameters

You can:

  • activate / deactivate the plugin
  • choose media directory
  • choose zoom level for maps

Types

You can:

  • add types by clicking "New type",
  • update by clicking thier names,
  • or delete them by check checkboxes and click "Remove selected"

Themes

You can add themes and subtheme. A subtheme is like a theme but with an existing theme as parent.

A parent theme can have a css class to display it in a particular way.

Sites

You can:

  • Add a site
  • Delete one or severals sites
  • Order sites by drag and drop. You must after that "Save order".
  • Change status for a site (or severals). Only administrator can change status. You can publish, unpublish or delete a site. By default a new site is pending. Deleted sites are not really deleted; they are only "visible" by administrator. That way, administrator can put it back online.

Installation

Installation tab gives you all tags defined by the plugin. You can also find examples of code to display themes list, sites list or a site in particular.

The plugin gives you three templates for easier installation:

  1. inc_thematics.html: use it in your _top.html template
  2. inc_thematic.html: use it in a page displaying sites for a selected theme
  3. inc_site.html: use it for a page displaying a site

To use theses three templates, you need to add the following codes depending on the page you want: 

      {{tpl:include src="inc_thematics.html"}}
    
      {{tpl:include src="inc_thematic.html"}}
    
      {{tpl:include src="inc_site.html"}}
