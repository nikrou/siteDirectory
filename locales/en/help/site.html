The following fields are mandatory: name, theme, description, long description, address, type, direction, town, latitude, longitude.

Usage

  • When selecting a theme, attached subtheme are populated dynamically.
  • The description is used in the "thematic.html" page
  • The long description is used for "site.html" page
  • The image field is optional. Images are saved in the media manager in a directory. You can choose that directory: go to parameters tab. When you edit a site, you can remove an image.
