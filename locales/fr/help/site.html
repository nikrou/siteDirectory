Les champs suivants sont obligatoires : nom, thème, description, description longue, adresse, type, cardinalité, ville, latitude, longitude. Les autres sont facultatifs.

Mode d'emploi

  • Lorsque vous choisissez un thème, les éventuels sous-thèmes associés apparaissent dynamiquement.
  • La description courte est affichée sur la page "thematic.html"
  • La description longue est affichée sur la page "site.html"
  • Le champ image est optionnel. Les images sont stockées dans le gestionnaire de médias, dans un dossier que vous pouvez choisir (cf onglet paramètres). Quand vous éditez un site ayant une image vous pouvez la supprimer du site (ne plus l'associer).
