Ce plugin vous permet de gérer une liste de sites.

Mode d'emploi

  1. Activer le plugin (onglet Paramètres)
  2. Ajouter des types (onglet Types)
  3. Ajouter des thèmes (onglet Thèmes)

Paramètres

Vous pouvez :

  • activer / désactiver le plugin
  • choisir le répertoire où seront stockées les images attachées aux sites
  • choisir le zoom pour les cartes

Types

Vous pouvez :

  • ajouter des types, en cliquant sur "Nouveau type",
  • les modifier, en cliquant sur leur nom,
  • ou les supprimer, en cochant la case associée et en cliquant sur "Supprimer les éléments sélectionnés"

Thèmes

Vous pouvez ajouter des thèmes ou des sous-thème. Un sous-thème est un thème dont le parent est un thème existant.

Un thème parent peut avoir une classe css particulière, ce qui permet de l'afficher de manière particulière.

Sites

Vous pouvez :

  • Ajouter un site
  • Supprimer un ou des sites
  • ordonner les sites par glisser/déposer. Il faut ensuite "Sauver l'ordre".
  • changer le statut de publication d'un site (ou de plusieurs sites). Seul l'administrateur peut changer le statut. Vous pouvez publier, mettre hors ligne, supprimer, et mettre en attente. Un nouveau site est par défaut en attente. Les sites supprimés ne sont pas réellement supprimés ; ils sont juste uniquement "visibles" par l'administrateur. Cela permet de le remettre en ligne si jamais on change d'avis.

Mise en place

L'onglet Installation donne l'ensemble des tags définis par le plugin et des exemples de code pour afficher la liste des thèmes, la liste des sites d'un thème donné et un site en particulier.

Le plugin fournit trois templates pour facilier l'intégration :

  1. inc_thematics.html : à utiliser dans _top.html
  2. inc_thematic.html : à utiliser sur la page affichant les sites d'un thème donné
  3. inc_site.html : à utiliser sur la page affichant un site

Pour utiliser ces trois templates, il faut les inclure dans une de vos pages avec respectivement les codes suivants : 

      {{tpl:include src="inc_thematics.html"}}
    
      {{tpl:include src="inc_thematic.html"}}
    
      {{tpl:include src="inc_site.html"}}
